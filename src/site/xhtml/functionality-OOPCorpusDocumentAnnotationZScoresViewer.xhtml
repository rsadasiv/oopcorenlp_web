OOPCorpusDocumentAnnotationZScoresViewer

Title: OOP Corpus Document Annotation ZScores Viewer
URI: OOPCorpusDocumentAnnotationZScoresViewer
Routing: servlets/OOPCorpusDocumentAnnotationZScoresViewer.java -> jsp/OOPCorpusDocumentAnnotationZScoresViewer.jsp -> js/OOPCorpusDocumentAnnotationZScoresViewer.js
Parameters: Corpus, Document, TargetCorpus, Annotation, AggregateName
Content type: text/html

Displays the z-score for each subannotation in the document against the selected target corpus in a vega-lite bar chart. Normalized is the most interpretable aggregate name.
Lists the Cosine Similarity score between the document subannotations and all corpora. Mouse-over anywhere in the chart activates the vega-embed action menu (circle with three dots on far right). Mouse-over a bar shows the data point value.
