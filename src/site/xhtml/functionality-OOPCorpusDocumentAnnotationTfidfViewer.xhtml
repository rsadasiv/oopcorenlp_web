OOPCorpusDocumentAnnotationTfidfViewer

Title: OOP Corpus Document Annotation Tfidf Viewer
URI: OOPCorpusDocumentAnnotationTfidfViewer
Routing: servlets/OOPCorpusDocumentAnnotationTfidfViewer.java -> jsp/OOPCorpusDocumentAnnotationTfidfViewer.jsp -> js/OOPCorpusDocumentAnnotationTfidfViewer.js
Parameters: Corpus, Document, TargetCorpus, Annotation
Content type: text/html

Displays the tf-idf score for each subannotation in the document against the selected target corpus in a vega-lite bar chart. Higher is better.
Mouse-over anywhere in the chart activates the vega-embed action menu (circle with three dots on far right). Mouse-over a bar shows the data point value.
