What does this do?

oopcorenlp_web takes in oopcorenlp analysis results and generates a web site.

Given oopcorenlp output as input, oopcorenlp_web provides the following:

  • Corpora
    • CorporaViewer
    • Batch info
    • CorpusCVScoresViewer
    • CorpusAnnotationCVScoresViewer
    • CorpusDocumentsViewer
    • Pipeline info
  • Read
    • TextViewer
    • TextOutOfPrintViewer
    • TextMediumViewer
    • Audio
  • Interact
    • OOPPullQuotesViewer
    • OOPActorsViewer
    • OOPFactCheckerViewer
    • OOPEditorViewer
  • Visualize
    • OOPStoryArcViewer
    • OOPCloudViewer
    • OOPStreamViewer
    • OOPStreamViewerTokens
  • Compare
    • OOPCorpusDocumentZScoresViewer
    • OOPCorpusDocumentAnnotationZScoresViewer
    • OOPCorpusDocumentAnnotationTfidfViewer
    • OOPLexiconViewer
  • Analyze
    • OOPDocumentViewer
    • OOP
    • StanfordBratViewer
    • Stanford
