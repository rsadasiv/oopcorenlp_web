What does this do?

oopcorenlp_web takes in oopcorenlp analysis results and generates a web site.

Given oopcorenlp output as input, oopcorenlp_web provides the following:

  • CorporaViewer
  • Batch info
  • CorpusCVScoresViewer
  • CorpusAnnotationCVScoresViewer
  • CorpusDocumentsViewer
  • Pipeline info
  • Plain Text
  • TextOutOfPrintViewer
  • TextMediumViewer
  • Audio
  • OOPPullQuotesViewer
  • OOPActorsViewer
  • OOPStoryArcViewer
  • OOPEditorViewer
  • OOPCloudViewer
  • OOPStreamViewer
  • OOPStreamViewerTokens
  • OOPDocumentViewer
  • OOP
  • StanfordBratViewer
  • Stanford
